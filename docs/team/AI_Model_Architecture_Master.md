# AI_ROUTER.md

# Objetivo

Este documento define qué modelo utilizar para cada tipo de tarea dentro de cualquier proyecto:

La regla principal es:

NO utilizar el mismo modelo para todo.

Cada modelo tiene una especialidad y debe utilizarse únicamente para aquello donde genera mayor valor.

---
