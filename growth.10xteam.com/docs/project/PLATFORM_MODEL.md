# PLATFORM MODEL - growth.10xteam.com

## Tesis del producto
El wizard ICP no solo genera materiales: es el gancho de entrada para convertir visitantes en leads calificados y, despues, en clientes con acceso a una plataforma privada.

## Flujo comercial propuesto
1. El usuario completa el wizard.
2. Se genera un documento atractivo de su ICP en version gratuita.
3. El documento se envia por correo a cambio de datos basicos de contacto.
4. El equipo usa ese lead para agendar cita y cerrar.
5. Si el cliente contrata, recibe login a su plataforma privada.
6. Dentro de la plataforma ve su ICP, materiales, contenido, seguimiento y futuras mejoras.

## Propuesta de valor por etapa
### Gratis
- Documento ICP visual y claro.
- Captura de email y datos del negocio.
- Base para agendar cita o iniciar seguimiento comercial.

### Cliente activo
- Login a plataforma privada.
- Historial de ICP y materiales.
- Generacion recurrente de contenido y activos.
- Seguimiento de tareas, aprobaciones y rendimiento.

## Modelo de negocio
### Opcion A - Entrada gratis + cierre de servicio
- El wizard y el reporte ICP sirven como lead magnet.
- Cierre por diagnostico, mejora web o paquete growth.
- Mejor para arranque rapido y caja temprana.

### Opcion B - Servicio inicial + plataforma recurrente
- Venta inicial de setup.
- Mensualidad por contenidos, seguimiento y optimizacion continua.
- Mejor para ingresos recurrentes y retencion.

### Opcion C - Hibrido recomendado
- Gancho gratis via ICP.
- Cierre de sesion/servicio inicial.
- Upsell a mensualidad con plataforma privada.
- Es la opcion mas viable para empezar sin friccion comercial.

## Componentes que debe tener la plataforma
- Login por cliente.
- Panel con ICP aprobado.
- Biblioteca de materiales y versiones.
- Calendario o tareas de contenido mensual.
- Reporte de rendimiento y aprendizajes.
- Espacio para solicitar nuevos temas o ajustes.

## Viabilidad
### Factores a favor
- El valor se entiende rapido: el cliente recibe algo util de inmediato.
- El contenido generado por el wizard reduce tiempo de venta.
- La plataforma crea retencion y justifica mensualidad.

### Riesgos
- Si el reporte gratuito no se ve suficientemente premium, baja conversion.
- Si el login llega demasiado pronto, puede aumentar friccion.
- Si la plataforma intenta resolver todo desde el inicio, el MVP se diluye.

## Regla de producto
Primero vender claridad y resultado. Despues vender continuidad y plataforma.
